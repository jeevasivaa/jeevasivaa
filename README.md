### 🛒 PhoneZone — Mobile Shop Management Platform

> **Full-stack production web app** for a Tiruchirappalli retail shop

```
Stack: FastAPI · Async SQLAlchemy · PostgreSQL (Supabase) · HTMX · Jinja2 · Pydantic v2 · Alembic · bcrypt
```

### 🔐 Smart Security Tag System — IoT Theft Detection

> **ESP32 firmware + Flask REST API + Web frontend** — a complete IoT pipeline

```
Stack: ESP32 · C (firmware) · Flask · SQLite3 (WAL mode) · Jinja2 · MOSFET power-gating
```

- Firmware: **deep sleep + EXT0 interrupt** wake for ultra-low power consumption
- **MOSFET power-gating** to cut peripheral power during sleep cycles
- Wi-Fi polling architecture with **command queue pattern** for reliable server communication
- Flask REST API with **WAL-mode SQLite3** for high-concurrency write safety
- Six-template Jinja2 frontend with dark industrial aesthetic
- Real-time alert dispatch and device state tracking
